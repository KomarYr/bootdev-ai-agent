# AI Agent Project

A toy AI agent built in Python for the Boot.dev course.

## What it does
- Uses an LLM to assist with code tasks
- Can call tools/functions
- Can inspect and improve a codebase

## Tech Stack
- Python
- OpenAI/Gemini API
- GitHub

## Setup
1. Clone the repo
2. Install dependencies
3. Add your API key
4. Run the agent

## Notes
This is a learning project and should be used carefully.
